# Adaptive Collaborative Correlation Learning-based Semi-Supervised Multi-Label Feature Selection (Access-MFS)
This repository provides the MATLAB implementation of the proposed method: Adaptive Collaborative Correlation Learning-based Semi-Supervised Multi-Label Feature Selection.
# Datasets
In the "Datasets" folder, we provide the dataset used in the experiments, langlog.mat. The remaining datasets can be downloaded from the sources provided in our paper.
# Code of Access-MFS
The code of the proposed Access-MFS can be found in the "Access-MFS" folder. Some explanations regarding the code are presented below:
1. The file "Access-MFS" is the core part of our algorithm, which contains the details of the optimization.
2. The file "ConstructSx" is used to update the instance similarity matrix.
3. The file "ConstructSf" is used to update the label similarity matrix.
# ML-KNN
In the "ML-KNN" folder, we provide the classification algorithm used in the experiments.
# Demo
The file "main" provides a usage example of our algorithm.
